Heart Attack Prediction

Data Source: Kaggle.com | Library: Pandas, Matplotlib, Seaborn

I have used the data from website Kaggle to predict the Heart Attack rate of the patient based on their biological dataset such as their age, gender, cholesterol amount etc.

After checking up with the data, see if there is any null values and see how many unique values for each columns, I have changed the categorical values into numeric values using mapping function to visualize those into plots. Below image shows the distribution of each categorical features.

From the graphs above, we were able to find out that all the features at least contribute to heart attack at certain level

After the visualization of the categorical features, I have decided to look into the numeric categorical features such as age, chol, and etc. Below graphs represent the distribution of each numeric features.

KDE Plot represents the data using a continuous probability density curve. The more two kde plots overlap, the less difference between those two
